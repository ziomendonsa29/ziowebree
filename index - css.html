Materia CSS

Cascading Style Sheets (CSS) mak língua estilu ne'ebé uza atu deskreve aprezentasaun dokumentu ne'ebé hakerek iha HTML ka XML (inklui dialetu XML hanesan SVG, MathML ka XHTML). CSS deskreve oinsá atu halo renderizasaun ba elementu sira iha screen, iha surat-tahan, iha diskursu, ka iha média seluk.

CSS nu'udar lian prinsipál iha internét nakloke no standardiza iha navegador web sira tuir espesifikasaun W3C nian. Antes nee, dezenvolvimentu ba parte oioin husi espesifikasaun CSS hala'o ho sinkroniku, ne'ebé permite atu halo versaun ba rekomendasaun sira ne'ebé foin lalais liu. Karik Ita rona ona kona-ba CSS1, CSS2.1, ka CSS3. Sei nunka iha CSS3 ka CSS4; maibé agora buat hotu mak CSS sein númeru versaun.
