Materia HTML

Lingua marka Hypertextu ka HTML mak lian marka padraun ba dokumentu sira ne'ebé dezeña atu hatudu iha navegador web. Dala barak nia hetan asisténsia husi teknolojia sira hanesan Cascading Style Sheets (CSS) no língua scripting hanesan JavaScript.

Broxador sira simu dokumentu HTML husi sítiu web ka husi armazenamentu lokál no halo tradusaun ba pájina web multimedia. HTML deskreve estrutura husi pájina web ida semantikamente no inisialmente inklui kues ba ninia oin.

HTML bele inkorpora programa sira ne'ebé hakerek iha língua eskritura hanesan JavaScript, ne'ebé afeta hahalok no konteúdu iha pájina web. Inkluzaun husi CSS define haree no arranju konteúdu. World Wide Web Consortium (W3C), eis manteinador ba HTML no agora mantein padraun CSS, enkoraja ona uza CSS hodi halo aprezentasaun esplikativu ba HTML dezde 1997.[2] Formatu HTML, ne'ebé koñese hanesan HTML5, uza atu hatudu vídeo no audio, liuliu uza elementu hamutuk ho JavaScript.
